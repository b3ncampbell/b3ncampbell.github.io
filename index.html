Deployment!

31st March 2017

Up until now this site only existed on my computer. Over the past couple of days I've been reading about the best way to get my work online, and have decided to use GitHub Pages as it is free, and I want to learn how to use Git also.

I've never used a command line before so I had to learn how that works, but after around an hour I managed to clone the repository, commit some changes and push them to GitHub.

I loaded up the site on my phone to see how it looks as this is how a lot of people are viewing the web nowadays and... it's not great. Just a scaled down version of everything and the header has been moved out of line somehow!

Next up will be making the site responsive, and learning more about Git, as I've read that it's widely used and a valuable skill to have when working with other developers.

Layout Adjustment

29th March 2017

Today I wanted to adjust the layout of the site to make navigation easier once there is more content. I figured it would be easier to do this now while there isn't too much content to deal with.

I've changed from the original layout which you can see here, to the layout that you are viewing now with a banner and navigation links to jump to specific blog posts.

Whilst making these changes I came across a few problems trying to get the main content boxes and the navigation box positioned how I wanted them, problems like this, and this. After some trial and error and a bit of googling I came to the conclusion that I needed to restructure the HTML of the site, putting most of the body in a wrapper and adding a few more divs.

Adding jQuery

27th March 2017

This will be a very short blog entry, but I want to keep a record of every change that I make as it helps me to retain information.

I've recently been learning some JavaScript and jQuery and wanted to add some to my site. I've put a link to the Google hosted jQuery library after reading that this can result in faster load times than if I were to host the library myself.

So far I've only added a page fade-in (which I'll probably remove later) to confirm that it is working, but it's a start.

The Beginning

26th March 2017

Hello, and welcome to my website. This site will serve as a diary of sorts, detailing new skills that I acquire in my journey of learning web development and programming.

As you can see the site is pretty basic at the moment, consisting of only HTML and CSS, but not for long. As I learn new skills I will implement them on this website and include a blog post about what I have learnt.

The next step is to add some JavaScript and jQuery to make the site more animated. I will then go about making the site responsive, for an optimal experience no matter where the site is being viewed.

My end goal is to begin a career in web development and this website is the start of my portfolio.
